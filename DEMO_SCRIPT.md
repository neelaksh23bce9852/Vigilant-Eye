*   **Action:** User toggles "Developer Mode" and loads "Vigilant Eye".

## Setup Instructions

1. Open Chrome > `chrome://extensions`
2. Enable "Developer mode" (top right).
3. Click "Load unpacked" and select the `vigilant-eye-extension` folder.
4. Open `demo-test-page.html` in the browser.
